# 🖥️ Dashboard UI

The dashboard provides a clean, real‑time view of system performance.

---

## Overview Card
Shows:
- Current power  
- Daily energy  
- System status  

---

## Daily Production Chart
Interactive chart showing:
- Power over time  
- Energy accumulation  

---

## Panel Grid
Displays:
- Per‑panel status  
- Output  
- Alerts  

---

## Alerts View
- Offline panels  
- Collector failures  
- Backup failures  

---

## Configuration Editor
Allows editing:
- Poll intervals  
- Thresholds  
- Panel definitions
